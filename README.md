# 🌌 Acerca del proyecto

**CarRentalsAgency** es un sistema de administración de renta de automóviles desarrollado para optimizar los procesos administrativos y operativos dentro de agencias de alquiler vehicular.

La plataforma permite gestionar:

- 🚗 Reservaciones de vehículos
- 👥 Registro de clientes
- 📅 Control de disponibilidad
- 💳 Administración de pagos
- 📊 Seguimiento operativo
- 📝 Gestión de membresías
- 📧 Comunicación con clientes
- ⭐ Retroalimentación y soporte

El sistema fue desarrollado pensando en:

- ⚡ Automatización administrativa
- 🌐 Acceso web centralizado
- 🧠 Arquitectura escalable
- 📱 Experiencia moderna
- 🔒 Gestión segura de datos

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/PlataformaGestionRentaAutomoviles.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd PlataformaGestionRentaAutomoviles
```

---

## 3️⃣ Instalar dependencias

```bash
composer install
npm install
```

---

## 4️⃣ Configurar variables de entorno

Crear archivo:

```bash
.env
```

Configurar:

```env
APP_NAME=CarRentalsAgency
APP_ENV=local
APP_KEY=
APP_DEBUG=true

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=carrentalagency
DB_USERNAME=root
DB_PASSWORD=
```

---

## 5️⃣ Ejecutar migraciones

```bash
php artisan migrate
```

---

## 6️⃣ Ejecutar servidor

```bash
php artisan serve
```

Aplicación disponible en:

```bash
http://localhost:8000
```
